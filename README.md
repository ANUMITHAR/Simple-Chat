
# Chat UI – Social Media Style

A simple **Chat User Interface** inspired by popular social media messaging apps (WhatsApp / Instagram DM).  
Built using **HTML and CSS only**, ideal for beginners practicing UI design.

---

## Features

- Social media–style chat layout
- Left and right message bubbles
- Chat header with user profile
- Message input box with send button
- Clean and modern UI
- Separate HTML and CSS files

---

## Technologies Used

- HTML5
- CSS3 (Flexbox)

---

## Project Structure

```

Chat-UI/
│
├── index.html

```

---

##  How to Run the Project

1. Download or clone the project
2. Open the folder in **VS Code**
3. Open `index.html`
4. Right-click → **Open with Live Server**  
   OR double-click `index.html` to open in browser
